# **Instalação**:

### **1 -** Clone o repositório.
```console
git clone https://github.com/aleffreitas/Healthy-Food
```

### **2 -** Entre na pasta.
```console
cd Healthy-Food/
```

### **3 -** Instale as dependências via **yarn**.
```console
yarn
```
### **4 -** Inicie a aplicação.

```console
yarn start
```
A aplicação roda em **http://localhost:3000/Healthy-Food**.
